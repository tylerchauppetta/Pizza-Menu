# Pizza-Menu

My Name: Tyler Chauppetta

Course: Software Development 1

CRN: 31032

This project is a nodel of a pizza ordering menu where the user can select crust type, topping type, and quantity of the pizza
and the program will display the price.

The project demonstrates the use of classes, abstract classes, and objects. Each class works with another to calculate a total
for the main application.

It is expected that a user can input simple number commands to navigate the menu, with the occsional alphabetical character to
either continue or terminate the program. The program is expected to output an accurate total based off specific parameters.
